myFlix (Angular)

  • TypeScript
  • Sass
  • Node.js

Much like my other myFlix project, this time created using Angular. The aim of this project was to build the client-side using Angular, based on its existing server-side code (REST API and database), with supporting documentation.   • Implemented services logic to make API calls
  • Imported models from Angular Material to serve to other components.
  • Created components and imported routing.
  • Added TypeDoc documentation and ensured code is commented adhering to best practices.

chat

  • React Native
  • Expo
  • Google Firebase
  • Gifted Chat

A mobile chat app built with React Native and developed with Expo. It is optimized for iOS and Android. Users can authenticate anonymously using Google Firebase authentication. Google Firestore Database is used to set up a real-time database. Messages are also stored locally on the user's mobile device via Async Storage, allowing users to access their messages offline. Users can send messages and share images and their locations.

What I did:

  • Made use of the React Native Gifted Chat library to create the UI.
  • Used Expo as a development environment to develop and test the app.
  • Set up Firestore database to store images, conversations, and real-time user data.
  • Utilized the React Navigation third party library to navigate between screens.

movie API

  • MongoDB
  • Node
  • Express
  • Mongoose
  • Postman

This is the back-end API to myFlix. This API provides endpoints to access a database to receive information about specific movies, and also allows a users to access a database that contains images, a synopsis and information about the director and movie genre. It has also endpoints for the user to register/deregister, log in/out, update their user profile, and add movies to a list of favorites.

What I did:

  • Created HTTP endpoints and route handlers with subsequent testing in Postman.
  • Created and populated a non-relational database with MongoDB.
  • Use Mongoose to build the Business Logic Layer linking the database from MongoDB to the server (and finally to the Web Browser).
  • Implemented authentication in Node.js/Express using Passport.
  • Implemented security measures for back-end including CORS and Bcrypt.
  • Hosted on MongoDBAtlas (DBaaS) and HEROKU (PaaS).
  • Troubleshooted the deployment process to HEROKU.
